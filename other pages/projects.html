SHC Rover Two-Month 2024

Project information

Date started: 9/2024

Date completed: 10/30/2024

SHC Rover Two-Month 2024 GitHub

This project involved programming a Raspberry Pi Pico microcontroller to interface with a Libre Renegade over serial using the ROS 2 language.

The microcontroller had to perform the following tasks
  • Read Accelerometer data and report it back to the Libre Renegade via serial and also store it on an SD card
  • Read enviromental data and report it back to the Libre Renegade via serial and also store it on an SD card
  • Take motor commands over serial and translate them into movements of the arm
  • Display the data received by the Libre Renegade and display it in a PyQt interface
